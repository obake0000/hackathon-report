CONCLUSION
いつもの作業
全日本AIハッカソンに絞ろう。

毎日Claudeで自動化している人にとって、3時間バトルは新しい挑戦ではなく、いつもやってる作業そのもの。

毎日ComfyUIで画像を量産してる人なら、3時間で動くデモを作るのは「今週のいつもの仕事」を90分前倒すだけ。
98% 自力で完結
相棒は「音楽」だけ補えばいい。

画像・3D・ゲーム・デザイン・プレゼン全部自力で出せる。残るのは音楽だけ。BGM・効果音が出せる人を1人連れれば装備は完成する。

普段ComfyUIで広告画像も作れてプレゼンも自分で回せる人なら、ハッカソンで探すのは「音作りが得意な相棒」1人だけ。映像が出せる人を探して時間を使う必要はない。
勝因は速さ
「動かす力」が「うまさ」に勝つ。

優勝作は技術がすごいのではなく「動いた・笑えた・拡散できた」。途中で止まったチームは審査外。

普段noteを毎日書いて出している人なら、完成度より「最後まで出す」癖がついていて、これだけで上位半分に入る。

💢 みんなここで止まる

課題 05

作りたいことが膨らんで、見せ場まで届かない。

結論として、対策は「戦場を絞る」「相棒を先に決める」「武器を固定する」「テンプレを持っておく」「練習で1回負ける」の5つでほぼ全部解ける。

▶ 行動1：戦場を全日本AIに絞る

3つあるけど、出るのは1つだけ。Web Speedは見送り、Impactは余力があれば、AIは本気。

全日本AI
いつもの武器で戦える
Impact Tokyo
学びと人脈用
Web Speed
フロント職人の世界
Reactでサイトを毎日チューニングしている人ならWeb Speedで戦える。普段Pythonで業務ツールを作る人なら、武器がそろうAIハッカソンの方が勝率は3倍くらい違う。

▶ 行動2：相棒は「音楽が出せる人」1人だけ

同じスキル（AI＋画像＋3D＋ゲーム）を持つ人は世の中に少ない。探すのに時間使うより、自分が薄い1点だけ補う人を1日で決める。

候補は1つだけ。BGM・効果音・SEが作れる人。Impact Tokyoは出ないので英語ピッチ要員は不要。プレゼンも自分で回せるのでMC役も不要。

声をかける場所はX、過去のハッカソンconnpass参加者、Comfy・Cursor界隈、音楽サークルの知人。1人めで断られたら3人めまで送る。

DTM経験のある知人や、Suno/Audio生成系で遊んでいるXの知人を1人引っ張れば、3時間で「動くデモ＋オリジナルBGM＋効果音」が揃う。他のスキルは自分で全部回す前提で動く。

▶ 行動3：使う道具を固定する

本番で迷わないため、武器セットを事前に決めておく。Cursorで雛形、Claudeで補強、ComfyUIで絵、Sunoで音、Sora 2で映像。

これ以外の道具は当日使わない。AIに指示してコードを書かせるやり方（バイブコーディング）に慣れるのが一番大事。

毎週ComfyUIで広告画像を作っている人なら、ハッカソンでもキャラ素材10分・背景10分で作れる。新しい道具を試すのは本番後に回す。

▶ 行動4：雛形を2つ作っておく

本番でゼロから作ると間に合わない。「1ファイルで動くゲーム」と「1ファイルで動くクイズ」のHTML＋JS雛形を事前に用意。

テーマ受けたらどちらかにテーマだけ差し替えて即動かす。動かない時間を作らない。

毎日ダッシュボードのHTMLを作っている人なら、ゲーム枠の雛形を1日で用意できる。クイズ枠は3択ロジック＋スコア表示の60行で十分。

▶ 行動5：練習エントリーで1回負ける

いきなり本気で出ない。6/20のオンライン回で1回出て、自分の弱点を洗う。7/25の東京回で本気を出す。

負ける目的は、本番の空気と時間配分を体で覚えること。優勝チームの動き方を観察するのも狙い。

投資ツールのバックテストを毎日やっている人なら「練習で1回流す」感覚は分かる。同じことをハッカソンでもやる。

▶ 当日3時間のタイムライン

0-15分 / コンセプト確定
15-110分 / MVP実装
110-150分 / 世界観統合
150-180分 / 提出

最初の15分でコンセプトを決め切る。残り95分でMVPを動かす。最後40分は映像・音・プレゼン台本。提出は最後の30分で必ず始める。
